Emojiq is a Slackbot that turns messages into emojis using modern JavaScript syntax in Node.js, including Promises and Fetch API, as well as Object-Oriented Design Patterns (OOP).

Technology used:

- DigitalOcean App Platform
- Express
- Github Deployment
- Node.js
- React
- Slack API
